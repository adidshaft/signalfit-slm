# Support

## Questions and usage help

Start with the [README](README.md), the [documentation index](docs/README.md),
and the [testing guide](docs/testing_guide.md). If the answer is not there,
open a GitHub Discussion when available or a focused issue using the relevant
template.

## Bug and model-behavior reports

- Use the bug form for reproducible failures in scripts, schemas, or tooling.
- Use the model-behavior form for grounding, field-binding, safety, or response
  quality problems. Supply only synthetic context data.
- Use the feature form for proposed capabilities or integrations.

## Security and privacy

Do not report vulnerabilities or private health data in public issues. Follow
[`SECURITY.md`](SECURITY.md) instead.

## Medical disclaimer

SignalFit-SLM is not a medical device and this repository does not provide
medical advice. For urgent symptoms or medical decisions, contact a qualified
health professional or local emergency service.
